title: Introduction
---

This is a fork of the original [Moe Counter](https://github.com/journey-ad/Moe-Counter). Moe Counter is a simple and customizable counter that allows you to create and display counters for your projects, blogs, or documentation. This fork adds additional features such as counter claiming, resetting, and deleting, allowing users to have more control over their counters.

You do not need to create an account to use the counter, but creating an account gives you access to additional features mentioned above. You can create an account by visiting the [website](https://count.timeworn.net/) and logging in with your GitHub account. Once logged in, you can create and manage your counters from the [dashboard](https://count.timeworn.net/dashboard).
