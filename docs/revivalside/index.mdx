[RevivalSide](https://github.com/MadlyMoe/RevivalSide) is a server implementation for a [certain urban graphic novel RPG](https://www.counterside.com/). This documentation will serve as a guide for setting up and using RevivalSide.

## Before You Start

<Callout type="warn">
  CounterSide is no longer available on the official app stores. See
  [announcement](https://www.counterside.com/notice/item/ct/en/tbl/notice/idx/4574/P1).
</Callout>

Before you begin, **ensure that you have CounterSide installed** on your device. You can download the game from the following app stores:

- [Steam](https://store.steampowered.com/app/1976440/CounterSide/)
- [Google Play Store](https://play.google.com/store/apps/details?id=com.studiobside.CounterSide)
- [Apple App Store](https://apps.apple.com/us/app/counterside/id1309394370)
