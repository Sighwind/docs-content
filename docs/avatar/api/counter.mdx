## Overview

This endpoint returns an image based on a seed/username. If the user has a custom avatar, it will return that instead. Otherwise, it will return a placeholder avatar.

## Endpoint

```
GET /avatar.timeworn.net/{seed}
```

## Example Requests

```
https://avatar.timeworn.net/demo
```

## Response

Returns an image representing the current count for the specified slug.
